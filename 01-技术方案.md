# 利尼尔CRM管理系统 — 宜搭技术方案

## 一、系统概述

- **平台**: 钉钉宜搭专业版（30人）
- **开发方式**: 宜搭低代码配置 + 连接器扩展
- **交付周期**: 25个工作日
- **需求来源**: 华泽合同附件《系统开发功能清单》

#### 3.2.2 客户公海管理

**实现方式：**
- 新建一张「公海规则配置表」
- 设置规则：客户状态为"流失"且超过N天未跟进的，自动标记为公海客户
- 设置定时触发（宜搭定时器），每天检查一次符合条件的客户
- 公海客户显示在「客户公海」视图，销售人员可按权限挑入
- 多公海：按区域维度配置不同的公海池

**数据表：公海规则配置**
```
┌─────────────────────────────────────┐
│ 公海规则配置                          │
├─────────────────────────────────────┤
│ 规则名称 (文本)                      │
│ 适用区域 (关联区域字典)               │
│ 未跟进天数阈值 (数字)                 │
│ 客户状态条件 (多选)                   │
│ 自动流转 (开关)                      │
│ 挑入权限 (多选: 全员/仅销售经理)      │
└─────────────────────────────────────┘
```

## 四、审批流程设计

### 4.1 审批流清单

| 审批流 | 触发条件 | 审批人 | 说明 |
|--------|---------|--------|------|
| 客户跟进审批 | 签到后自动发起 | 直属上级 | 快速审批 |
| 报价单审批 | 金额<¥10K | 销售主管 | |
| | 金额¥10K~¥50K | 销售主管+经理 | 自动多级 |
| | 金额>¥50K | 销售主管+经理+总经理 | |
| 订单审批 | 同报价单分层逻辑 | 同上 | |
| 费用报销审批 | 金额<¥1K | 直属上级 | |
| | 金额>¥1K | 上级+财务 | |
| 采购申请审批 | 金额<¥5K | 仓库主管 | |
| | 金额>¥5K | 主管+经理 | |
