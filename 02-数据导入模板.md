# 数据导入模板 & 产品分类设计

## 一、产品线结构（从目录中识别）

从产品目录来看，利尼尔的产品体系如下：

### 四大系列

```
利尼尔工业装备
├── SG系列 — 高扭矩型滚珠线性滑轨 (High Torque Ball Guide)
│   ├── SGH-A / SGH-AL   (法兰型/加长法兰型)
│   ├── SGS-A / SGS-AS   (滑块型/短滑块型)
│   ├── SGH-B / SGH-BL   (下锁法兰型)
│   └── SGS-B / SGS-BS / SGS-BL (下锁滑块型)
│
├── SM系列 — 高静音重负荷型滚珠线性滑轨 (High Precision Heavy Duty Ball Guide)
│   ├── SMH-A / SMH-AL
│   ├── SMS-AS / SMS-A
│   ├── SMH-B / SMH-BL
│   ├── SMS-BS / SMS-B
│   └── SMH-K (下锁式)
│
├── SR系列 — 滚柱型直线导轨 (Roller Type Linear Guide)
│   ├── SRH-A / SRH-AL
│   ├── SRH-B / SRH-BL
│   └── SR-K (下锁式)
│
└── SV系列 — 微型滑轨 (Miniature Linear Guide)
    ├── 非互换型
    └── 互换型
```

每款产品还有以下维度：
- **尺寸规格**：15/20/25/30/35/45 等宽度系列
- **精度等级**: C / H / P / SP / UP
- **预压**: V0(轻预压) / V1(中预压) / V2(重预压)
- **滑块数量**: 1~4个/支
- **导轨长度**: 按客户需求定制
- **是否带防尘钢带**: Y/N
